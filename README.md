# AI Spam Email Classifier

An AI-powered spam email classification web application built using React, TypeScript, Vite, Tailwind CSS, and Ant Design. The application allows users to paste email content and instantly determine whether the message is Spam or Not Spam using a machine learning backend API.

---

## Features

- Spam email prediction
- Clean and responsive UI
- Fast API integration
- AI/ML-powered classification
- Built with TypeScript support

---

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Ant Design
